Multimodal Classification for #MeToo Movement

IEEE BigMM 2020 Grand Challenge - 1st Rank

Developed a state-of-the-art multimodal classification model using ULMFiT LLM for analyzing #MeToo related tweets. The model successfully classified multiple linguistic aspects including Stance, Relevance, Hate Speech, and Sarcasm.

Musical Instrument Classification using Spectral Features

IEEE SPIN 2020

Developed an innovative approach for real-time musical instrument classification using spectral features. Implemented and compared various ML classifiers, with SVM + Ensemble achieving the best performance.
